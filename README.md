# SinsaMarket
![image](https://user-images.githubusercontent.com/58693617/146855613-b1bad599-36e6-46d5-a2c4-7ae4be45f16b.png){: width="100" height="100"}

## 주요 기능
* 회원가입, 로그인
* 회원가입 시 아이디, 이메일 중복 체크 기능을 수행한다.
* 판매 물품을 카테고리 별로 검색하는 기능
* 판매자가 판매하는 물품 조회 기능


## 사용기술
JAVA 11
Spring Boot 2.6.1
Gradle Project
Spring Security
Web Socket
JUnit
MySQL
AWS EC2
AWS RDS
